SPEECH EMOTION RECOGNITION

PROJECT DESCRIPTION

This Speech Emotion Recognition (SER) project uses deep learning to classify emotions from audio recordings. Utilizing the Toronto Emotional Speech Set (TESS) and advanced preprocessing techniques like MFCC feature extraction, the project employs an LSTM neural network to accurately detect emotions such as anger, happiness, and sadness in speech. The model is designed to help in various applications, including emotion analysis in customer service and mental health monitoring.
